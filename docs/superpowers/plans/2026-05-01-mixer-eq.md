**Goal:** Make each deck's `Hi`, `Mid`, and `Low` knobs control a real three-band EQ on the playing track audio.

**Architecture:** Add a small pure helper for knob-to-dB mapping, plus a focused Web Audio helper that creates one EQ chain per deck. Update `src/App.tsx` to route each hidden `<audio>` element through that chain, keep crossfader/level control on the output gain node, and update filter gains whenever mixer knob state changes.

**Tech Stack:** React 19, TypeScript/JS modules, Vite, Web Audio API, node `assert`

### Task 2: Implement EQ mapping and audio-chain helpers

**Files:**
- Create: `src/eq.js`
- Modify: `src/mixer.js`
- Test: `src/eq.test.mjs`

- [ ] **Step 1: Write minimal implementation**

```js
const clampKnob = (value) => Math.min(Math.max(Number.isFinite(value) ? value : 50, 0), 100);

export const getEqGainDbFromKnobValue = (value, maxGainDb = 18) => {
  const normalized = (clampKnob(value) - 50) / 50;
  return Number((normalized * maxGainDb).toFixed(3));
};
```
